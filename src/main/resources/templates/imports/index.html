Import de données

Reprise des relevés de terrain depuis un fichier Excel (.xlsx)

Seules les données d'observation sont importables : pesées, mortalités et distributions d'aliment. Les ventes, les achats et les lots restent saisis dans l'application, où leur cycle de vie et leurs effets de bord sont contrôlés.
Chaque import se fait en deux temps : le fichier est d'abord analysé et vous est présenté, rien n'est enregistré tant que vous n'avez pas confirmé.

Comment procéder
  1. Téléchargez le modèle : il contient les bons en-têtes, une ligne d'exemple, et des listes déroulantes alimentées par vos lots actifs et vos aliments. Vous n'avez donc pas à recopier un code à la main
  2. Remplissez une ligne par relevé, puis enregistrez le fichier au format .xlsx
  3. Déposez-le ici : chaque ligne est vérifiée et vous voyez le résultat avant tout enregistrement
  4. Confirmez. L'import est tout ou rien : si une ligne est refusée à l'enregistrement, aucune n'est écrite
